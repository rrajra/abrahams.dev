Scheduely

Description

After witnessing my managers struggle to create schedules and losing hair over it, I decided to make a free app to do it for you! Scheduely is a WPF Application written in C#. You can easily add or remove employees, and edit their availability. For easy usage, the program will write a file to disk so data will not have to be inputted each time. Once done editing, the user can click on "Create Schedule" and the program will create a schedule that best works for the employees.
